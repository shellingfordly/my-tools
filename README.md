# Tools

这是一个个人向的用于一些文件格式转换的工具，暂时只做了视频格式转换，后续会持续添加功能，可以关注本项目。

## Feature

1. 视频格式转换

包括视频转 gif/mp4/avi/webm/mpeg/flv，使用 ffmpeg 库，转换格式为 ffmpeg 支持的格式
